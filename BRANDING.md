GymZ embodies the Steve Jobs philosophy of innovation:

> "Innovation is not about adding more. It's about removing everything that doesn't matter."

1. **Simplicity** - Remove complexity, add clarity
2. **Elegance** - Minimal, cinematic, effortless design

4. **Empowerment** - Where gyms run themselves

- Break into new lines. Create rhythm.
- Use em dashes ‚Äî they create pause and impact.
- Italic quotes for emphasis
- No fluff. Every word earns its place.

## Visual Identity

### Design Principles

1. **Minimal** - Clean, focused interface
2. **Cinematic** - Large, impactful visuals
3. **Effortless** - Intuitive without trying
4. **Premium** - Every pixel breathes simplicity
5. **Invisible** - Great design you feel, not see

### Typography Hierarchy

**Headlines:** Large, bold, tracking-tight
- H1: 4xl ‚Üí 8xl (responsive)
- H2: 3xl ‚Üí 5xl
- Use line breaks for rhythm

**Body Copy:** Relaxed, readable
- Base: lg ‚Üí xl
- Muted foreground for secondary
- Generous line height

### Color Philosophy

**Background:** Pure simplicity
- Light mode: Clean whites
- Dark mode: Deep, rich blacks

**Accents:** Subtle gradients
- Primary: Blues, purples
- Hover states: Gentle transitions
- Never overwhelming

### Spacing

**Generous White Space:**
- Section padding: py-20 ‚Üí py-32
- Max-widths for readability
- Breathing room between elements

**Responsive Scaling:**
- Mobile: Compact but not cramped
- Desktop: Luxurious spacing

- Then specialization
- Use as rhythm throughout
